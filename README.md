Privacy Policy for Video Speed Control Plus

Video Speed Control Plus does not collect, store, sell, or share personal user data.

The extension only uses Chrome storage to save local user preferences such as selected language, theme, floating controller position, and optional per-site video speed settings.

The extension reads video elements on web pages only to change HTML5 playback speed according to the user's choice. No browsing history, video content, personal information, analytics data, or account information is collected or transmitted.

All extension code runs locally in the browser. The extension does not use remote code.
